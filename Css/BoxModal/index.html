Box modal

There are two types of box modal,

  1. content-box (width & height is only content area)
  2. border-box (width & height is content area + padding + border)

content-box

width: 250px height: 250px margin: 10px padding: 10px border-width: 5px border-radius: 10px outline-width: 1px outline-offset: 2.5px

border-box

width: 250px height: 250px margin: 10px padding: 10px border-width: 5px border-radius: 10px outline-width: 1px outline-offset: 2.5px

Note: Margin between the two box collapse into one. i.e 10px + 10px didn't add up to 20px instead collapse into 10px (largest one)
