Grid

It is used to create 2D layout by defining the rows & columns.

.parent { display: grid; }

This makes the .parent a grid container and it's direct children a grid items.

layout defination property

[container]

  • grid-template-rows
  • grid-template-columns
  • grid-template-areas

layout defination property

[item]

  • grid-row
  • grid-column
  • grid-area

Template Row & Column

Note: i. by default, on setting display: grid; it will create 1 column with as many rows required. ii. also, margin collapsing will not longer work!
