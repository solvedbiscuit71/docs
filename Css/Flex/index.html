Flex

it's a 1D layout for creating rows and columns

.parent { display: flex; }

This will make the .parent a flex container and it's direct child a flex items

Default:

  • direction will row [can be changed by flex-direction on parent]
  • flex-basis will look for width as it's set to auto
  • flex-shrink will be 1 and flex-grow will be 0
