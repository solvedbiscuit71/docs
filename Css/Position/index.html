Position

There mainly three type of positioning values

  • static (default)
  • relative
  • absolute

Positioned Element

An element is said to be positioned element if it's position property is not set to static.

  • element come on-top of the layout. (z-index increased)
  • it's position in screen remains unchanged until set manually.
