Display Property

There are mainly three types of display element

  • inline - padding,border,margin(inline)
  • block - padding,border,margin(both),width,height
  • inline-block - padding,border,margin(both),width,height

There major difference between a inline and inline-block are

  • inline-block support width and height whereas inline doesn't support.
  • inline-block support marging in both axis whereas inline support only in the inline axis.
