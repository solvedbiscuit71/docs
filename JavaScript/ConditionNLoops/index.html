Truthy & Falsy

An expression can result in any of the following, there are classified as Truthy & Falsy. if resultant is 'Truthy' it's consider to be true, else false

  • Truthy
    1. true
    2. number ( exclude 0 )
    3. string ( exclude '' )
    4. Array ( include [] )
    5. Object ( include {} )
    6. Function
  • Falsy
    1. false
    2. undefined
    3. null
    4. 0
    5. '' empty string

Logical operator & it's flow

There are three types of logical operator,

  1. !
  2. &&
  3. ||

Fallback Case

Something we want to use an fallback value for an variable,
Ex: we have a array in Database (but it may be null) in that case we need an fallback.

/* Assume, localStorage.getItem('array') as your database (it may return an array if present else null) if we have array then take it else set to [] */ let array = localStorage.getItem('array') || []

Condition Case

Sometimes we want do something but if some condition is satisfied.
Ex: you want to call a function if some value turned out to be true.

/* Assume, isComplete -> variable for checking callback() -> function need to call if isComplete is 'Truthy' then, return callback() */ return isComplete && callback()

Ternary Operator

This is a classic if-else substituent for return value statement.
Ex: if we want to find whether an number is even or odd.

let evenOrOdd = num % 2 === 0 ? 'even' : 'odd';
