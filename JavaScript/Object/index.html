Object

Every time (except primitive data types) are considered an object in javascript. any object are considered Truthy.

  1. Array []
  2. Object {}
  3. Function (){}
  4. Instance of a class

Array:

An array is a collection of heterogeneous datatypes.

Basics

  • arr[i] -> indexing starts from 0
  • arr.push(x) -> add x to end of array
  • arr.pop() -> remove last item
  • arr.unshift() -> add x to start of array
  • arr.shift() -> remove first item
  • arr.slice(start,end) -> slice the array (negative indexing allowed)

Advanced

  • arr.forEach() -> to iterator every item
  • arr.find() -> to find the first occurance
  • arr.findIndex() -> to find the index of first occurance
  • arr.filter() -> to filter the array based on condition
  • arr.map() -> to create another array based on current array
  • arr.reduce() -> to compute some value based on items in array

Object:

an object is a key-value pair (like, dictionary in python). in fact an array is an instance of Object.

Methods

  • obj.keys() -> returns an iterator of key
  • obj.values() -> returns an iterator of value
  • obj.entries() -> returns an iterator of [key,value] pair

Object Destructuring:

Object Destruturing is an method to destructure the object into individual pieces.
Here, we use the ... seperate operator with brackets to destructure it.
