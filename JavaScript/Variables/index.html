Primitives

There are 7 different primitive data types in javascript.

  1. bigint
  2. boolean (true, false)
  3. null
  4. number (0, 1, -3.14, 1.2e-10)
  5. string ('java', "script")
  6. symbol
  7. undefined

Variables

There are three way to declare a variable,

  1. var
  2. let
  3. const

Different between var, let and const

var

In var, we can declare and assign a variable and then reassign it later.

  • function scope
  • allows redeclare
  • allows hoisting

let

In let, we can declare and assign a variable and then reassign it later.

  • block scope
  • doesn't allow redeclare
  • doesn't allows hoisting

const

In const, we have to declare and assign a variable at the same line and it can't be reassigned.

  • block scope
  • doesn't allow redeclare & reassign
  • doesn't allows hoisting

Identifier

Variables are container that store data in them. There are some rules in naming a variable in javascript,

  • it should start with either,
    1. alphabet (A-Z a-z)
    2. underscore (_)
    3. dollar sign ($)
  • followed by,
    1. alphabet (A-Z a-z)
    2. numbers (0-9)
    3. underscore (_)
    4. dollar sign ($)

var $Name123; var _age_of_person; var 0address;

camelCase

This is a naming convention used it javascript (not mandatory).

  • The first letter of the first word should be lowercase
  • The following first letter of the remaining word should be uppercase
Ex:
var userName; var onClick;
