Forms

Form are essential part for getting data from the user.

Thing to note in form

  • Every input element in a form has it's own state variable
    • text input -> value
    • checkbox & radio -> checked
  • We can set & get the state variable by {element}.value
  • We can add oninput event listener to check for changes in state variable
  • Note: the event parameter from oninput will have the input element at event.target
