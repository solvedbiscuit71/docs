Function

Function are reusable block of code. there are lot of different ways to defined a function in javascript.

  • Function Declaration
  • Function Expression
  • Immediately Invoked Function Expression
  • Arrow Function

Function Declaration

function say(name) { return `my name is ${name}`; } say('praveen') // my name is praveen

Function Expression

/* Note: the function without a name is called Anonymous Functions. */ var say = function (name) { return `my name is ${name}`; } say() // my name is undefined

Immediately Invoked Function Expression

/* These function are immediately called on execution. (function() { body })(); */ const num = -function () { return 5; }(); console.log(num) // -5

Arrow Function

const say = (name = 'solvedbiscuit71') => { return `my name is ${name}` } say() // my name is solvedbiscuit71 /* You can also remove the () & {} if you have - one argument - only return statement. */ const say2 = name => `my name is ${name}` say2('teju') // my name is teju
